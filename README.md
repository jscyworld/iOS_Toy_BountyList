# iOS_Toy_BountyList
iOS Toy Project: BountyList

1. BountyInfo를 UITableView를 통해 보여주기
2. Custom UITableViewCell을 만들어 원하는 모양으로 보여주기
3. DetailViewController를 사용하여 Modal로 크게 보여주기
4. MVVM 패턴으로 Re-factoring
